### 🎯 功能演示流程

1. **启动应用** → 进入首页全屏图片浏览
2. **上滑** → 查看下一张随机图片
3. **下滑** → 查看上一张历史图片
4. **长按** → 弹出操作菜单 (收藏/下载)
5. **左滑** → 进入收藏页面
6. **点击缩略图** → 查看收藏大图
7. **长按缩略图** → 删除收藏
8. **右滑** → 返回首页
